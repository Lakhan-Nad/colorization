# Colorization

A flask web service to colorize gray scale images using Convolutional Neural Networks

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Open app.py and change UPLOAD_FOLDER
```python
UPLOAD_FOLDER = '/path/to/files'
```
Run the program using 
```bash
python3 app.py
```
